# About Me

Hello, I'm Hchyeria. 
热爱生活，喜欢 Code、读书、音乐、乐器、电影、视觉艺术、音乐剧，还有健身。

闲下来的时候会研究一些算法问题，会阅读一些源码。目标是成为优秀的 Software Engineer。

站名叫做 Avalon，意为遗世独立的理想乡。

**喜欢二次元。**但目前是佛系退坑状态，已经很久没有追番了。现在来看就是曾经有过的一段中二时期hhh。本命是吾王阿尔托莉雅。喜欢吉卜力，京阿尼，魔圆，巨人，石头门，空境&魔夜，少女番，喜欢百合作品。

**喜欢读书。**尽可能的涉及多个领域。

**喜欢电影。**喜欢看一些文学作品改编的电影。导演比较喜欢大卫·林奇、韦斯·安德森等等。喜欢动画电影，喜欢歌舞片、文艺片。喜欢看蛮多类型的，除了恐怖片，下真的打死也不看恐怖片。**喜欢追一些口碑不错的国外剧。**年代久远也没关系，但是不喜欢太黄暴的。主要是喜欢磕 CP，无论性别，只要是真爱。

理想生活是每天能有时间读书、练琴、看电影、磕CP、健身、写代码。每年都能抽空出国旅游。
